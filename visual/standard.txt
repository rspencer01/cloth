All inputs should follow the following format:

line 1   :  Number of frames
line 2   :  Number of points
line 3   :  Number of lines
line 4   :  Pairs of integers, (a,b) indicating that a is connected to b
line 5.. :  Data line pairs
 +line A :  Positions, space separated.  X, Y, then Y for each point
 +line B :  Integer, number of integers to follow.  Id's of links to not draw, in same index as was given


All data should be in the range [-1 , 1] to be viewed
